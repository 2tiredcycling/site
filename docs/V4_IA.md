# V4.0 站点信息架构（IA）

适用版本：`V4.0`

更新时间：2026-03-19

## 1. 一级导航

1. 首页 `/`
2. 社团介绍 `/about`
3. 管理团队 `/team`
4. 活动 `/events`
5. 路线 `/routes`（GPX 子模块）
6. 反馈 `/feedback`
7. 联系 `/contact`

## 2. 页面树

## 2.1 公开前台
- 首页 `/`
- 社团介绍 `/about`
- 管理团队 `/team`
- 活动列表 `/events`
- 活动详情 `/events/<id>`
- 活动报名 `/events/<id>/register`
- 报名成功 `/events/<id>/register/success`
- 路线中心 `/routes`（保留现有首页路线查询入口）
- 路线详情 `/routes/<id>`
- 网站反馈 `/feedback`
- 联系我们 `/contact`

## 2.2 后台 manage
- 总览 `/manage`
- 内容管理 `/manage/content`
- 公告管理 `/manage/announcements`
- 静态页管理 `/manage/pages`
- 首页配置 `/manage/homepage`
- 活动报名管理 `/manage/event-registrations`
- 现有模块保留：路线、活动、反馈、安全、统计、用户、审计

## 3. 入口与返回路径

- 首页进入：导航直达全部一级栏目
- 活动详情返回：活动列表
- 报名成功返回：活动详情 / 首页
- 后台内容编辑返回：对应列表页
- 后台统一保留“返回总览”

## 4. 权限视图（页面级）

- 公开可见：前台全部页面（报名提交除限流外不要求登录）
- 登录管理员可见：`/manage/**`
- 仅有内容权限可见：内容管理、公告管理、静态页管理、首页配置
- 仅有审核权限可见：报名管理、反馈审核
- 仅有用户管理权限可见：管理员账号管理

## 5. 导航设计约束

- 手机端一级导航不超过两行
- 每个页面至少一个明确返回入口
- 不新增深层菜单（最多三级）
